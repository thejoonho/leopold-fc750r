# Custom Bluetooth Keyboard: Leopold FC750R

🛠️ Converted a commercial wired keyboard (Leopold FC750R) into a Bluetooth keyboard. I designed the Bluetooth Keyboard's schematic and PCB layout, assembled the keyboard, and tested the complete system. Check out the project's <a href="https://docs.google.com/document/d/1T1kuHEfLhl1W9Vp5NQ1bjV8UXPvdlBXKJn0wyxQ07zg/edit?usp=sharing">**official documentation**</a>.
